Lista Descritiva

Listas Não Ordenadas
As listas não ordenadas inserem marcadores na frente de cada item.
Listas Ordenadas
As listas ordenadas inserem números ou letras na frente de cada item.
Listas Descritivas
As listas descritivas são diferentes das outras listas. Cada item de uma lista descritiva possui dois componentes: um tópico e uma descrição.
